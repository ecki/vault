**`write_data (string : <required>)`**

Data in the form `"<key>=<value>"` for the CLI request. Repeat inline
`write_data` items as needed to provide all necessary data. Vault also accepts
[key/value pairs](/vault/docs/commands#passing-data) as JSON objects passed in a
file or `stdin`.

**Example**: `credential_type="iam_user"`
