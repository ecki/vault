# `version`

Print version information for the **local CLI**. For version information related to
the Vault instance, use
[`vault version-history`](/vault/docs/commands/version-history).

<CodeBlockConfig hideClipboard>

```shell-session
$ vault [version | -v | --version]

$ vault version [-help | -h]
```

</CodeBlockConfig>

## Description

`vault version` prints version details for the local Vault CLI.

- Version number
- Github commit ID for the associated build
- Timestamp for the associated build

The output syntax is:

```bash
Vault <version> (<build_commit_ID), built <build_timestamp>
```

Vault also aliases the `version` command with CLI-level flags: `-v` and
`--version`

## Examples

Print CLI version information:

```shell-session
$ vault version

Vault v1.17.5 (4d0c53e84094b8017d32b6e5b7f8142035c8837f), built 2024-08-30T15:54:57Z
```

Print CLI version information with the `--version` flag alias:

```shell-session
$ vault --version

Vault v1.17.5 (4d0c53e84094b8017d32b6e5b7f8142035c8837f), built 2024-08-30T15:54:57Z
```
