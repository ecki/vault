# `status`

Print the current state of your Vault instance.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault status <flags>

$ vault status [-help | -h]
```

</CodeBlockConfig>

## Description

`vault status` returns seal and state information for the target Vault instance.
The specific information returned depends on whether the instance is currently
sealed or unsealed.

Information                                         | Sealed | Unsealed
--------------------------------------------------- | ------ | --------
Sealing algorithm for the cluster                   | YES    | YES
Initialization status for the cluster               | YES    | YES
Seal status for the cluster                         | YES    | YES
Number of unseal keys issued during initialization  | YES    | YES
Number of unseal keys required to unseal            | YES    | YES
Number of unseal keys provided so far               | YES    | **NO**
One-time decryption key (only during unsealing)     | YES    | **NO**
Vault version and edition                           | YES    | YES
Vault binary build timestamp                        | YES    | YES
Backend storage type                                | YES    | YES
Resource name for the cluster primary               | **NO** | YES
Resource ID for the cluster primary                 | **NO** | YES
If the cluster has high availability enabled        | YES    | YES
IP address for the cluster primary                  | **NO** | YES
If the cluster has high availability active         | **NO** | YES
Latest HA activation timestamp                      | **NO** | YES
Storage and logging index data                      | **NO** | YES

`vault status` also returns different exit codes depending on the seal state of
the Vault instance.

Exit code | Description | Seal state
--------- | ----------- | ----------
0         | Success     | Unsealed
1         | Error       | Unknown
2         | Success     | Sealed

HA Enabled              true

Raft Committed Index    29251856
Raft Applied Index      29251856
