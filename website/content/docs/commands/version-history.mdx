# `version-history`

Print the install history of your Vault instance.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault version-history <flags>

$ vault version-history [-help | -h]
```

</CodeBlockConfig>

## Description

`vault version-history` prints the installation history of your Vault instance
in chronological order. History information includes:

- Version number
- Install timestamp
- Build date for the associated version


<Tip title="Related API endpoints">

  ReadVersionHistory - [`LIST:/sys/version-history`](/vault/api-docs/system/version-history)

</Tip>

## Examples

Print the version history:

```shell-session
$ vault version-history

Note:
Use of this command requires a server running Vault 1.10.0 or greater.
Version tracking was added in 1.9.0. Earlier versions have not been tracked.


Version  Installation Time     Build Date
-------  -----------------     ----------
1.17.6   2024-10-06T05:33:45Z  2024-09-24T19:48:48Z
1.18.1   2024-11-16T01:15:59Z  2024-10-29T14:21:43Z
```
